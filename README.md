# ⚽ A.R.E.S. - Advanced Real-time Evaluation System

El Cerebro del Fútbol - Análisis predictivo, fatiga, tarjetas y más.

## 🚀 Características
- Predicción de resultados con xG
- Monitor de fatiga en tiempo real
- Predicción de tarjetas amarillas/rojas
- Historial de predicciones
- Exportación a PDF
- Análisis Champions League

## 🔧 Variables de Entorno
- `API_FOOTBALL_KEY`: Tu clave de API-Football
